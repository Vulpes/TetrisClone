# TetrisClone
Projeto feito utilizando a engine Unity3D e a linguagem C#. 

Este projeto foi feito com o objetivo de aumentar meus conhecimentos com o desenvolvimento de jogos 2D na unity. Este é um clone do clássico jogo Tetris, sua pontuação é calculada seguindo a regra clássica do Tetris.

Este jogo foi publicado na plataforma do itch.io, link: https://vulpesm.itch.io/tetris-clone
